## Tech Stack

**Client:** HTML, CSS, JS

**Server:** Node


## Run Locally

Clone the project

```bash
  git clone https://github.com/Mehmet1414/weather-app
```

Go to the project directory

```bash
  cd weather-app
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```


## Running Tests

To run tests, run the following command

```bash
  npm run test
```
